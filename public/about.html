Features

List View

The list view feature in Aventuras PR provides users with a convenient way to explore and browse through various activities, restaurants, and stays in Puerto Rico.

Map View

The map view feature provides users with an interactive map interface that displays various points of interest and locations in Puerto Rico. Users can easily navigate the map, zoom in and out, and explore different areas of interest.

Categorys

Aventuras PR is a comprehensive web application that offers users the convenience of searching for restaurants, lodging, and entertainment options all in one place. With this feature, users can easily find and explore various establishments and activities in Puerto Rico without the need to visit multiple websites or platforms.

Developer

Hello, I'm Lizbeth Garcia, the developer behind this web application. The inspiration for creating this application came from the common challenge we face when planning trips. It can be frustrating to switch between multiple pages to gather all the necessary information. That's why I wanted to create a single platform where users can find everything they need for their travel research.
